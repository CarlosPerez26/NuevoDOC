# NuevoDOC 🚀

Este repositorio contiene la configuración centralizada de mi entorno de desarrollo, incluyendo dotfiles, especificaciones técnicas y habilidades personalizadas para agentes de IA.

## 📂 Estructura del Proyecto

*   **`.agents/`**: Colección de *skills* y configuraciones para agentes de IA (ej. Neovim especializado).
*   **`dotfiles/`**: Archivos de configuración para herramientas clave:
    *   `init.lua`: Configuración avanzada de NeoVim (v0.11.4+).
    *   `.zshrc`: Personalización de la Shell con Zsh y Powerlevel10k.
    *   `.tmux.conf`: Configuración para multiplexor de terminales.
*   **`Dockerfile`**: Definición de la imagen base para el entorno de desarrollo.
*   **`mi-stack.md`**: Documentación detallada de las tecnologías (Python, JS/TS, Svelte, NeoVim).

## 🛠️ Tecnologías Principales

*   **Backend:** Python 3.10+, FastAPI, Uvicorn.
*   **Frontend:** Node.js LTS, TypeScript, SvelteKit, TailwindCSS.
*   **Herramientas:** NeoVim, Tmux, Zsh, Bun.

## 🚀 Cómo usar este repositorio

Este repositorio está diseñado para ser la fuente de verdad al configurar nuevos contenedores o entornos de desarrollo.

1.  Clona el repositorio: `git clone https://github.com/CarlosPerez26/NuevoDOC.git`
2.  Sincroniza los dotfiles con tu home directory.
3.  Utiliza el `Dockerfile` para construir una imagen de desarrollo consistente.
